# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-11-02
### Added
- Material UI + i18next frontend featuring dashboard, activity, and settings screens.
- Go backend utilities for OpenAI-compatible connectivity checks and persistent user settings.
- Sample activity feed generators and overview aggregation endpoint.
- Comprehensive documentation set (README, CONTRIBUTING, GOVERNANCE, ROADMAP, SECURITY, SUPPORT).
- GitHub Actions workflows for build/test, release packaging, and security scanning (SAST, DAST, SCA).
- Dependabot configuration for npm, Go modules, and GitHub Actions.

### Changed
- Replaced the default Wails Preact template with the Enigma Desktop Copilot experience.

### Security
- Added guidance for CodeQL, secret scanning, and dependency auditing.
